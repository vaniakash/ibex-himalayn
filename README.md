# IBEX Trekking

IBEX is a premium Indian trekking company website. The brand sits between rugged expedition and refined editorial — inspired by Himalayan wilderness.

## Tech Stack
- **Frontend**: Next.js 14 (App Router), React
- **Backend**: Express.js, Node.js (Pending)
- **Database**: MongoDB (Pending)
- **Payments**: Razorpay (Pending)

## Local Development
To run the frontend:
```bash
cd client
npm install
npm run dev
```

Visit `http://localhost:3000` to view the application.
